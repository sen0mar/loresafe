# ThreadSync Codex Skills

This folder contains three Codex-compatible skills:

- `tester` — adds focused tests for the feature just built.
- `commenter` — adds concise comments to the non-obvious parts of the feature just built.
- `cleaner` — performs precise cleanup refactors on user-directed ThreadSync code.

## Install

Copy the skill folders into a Codex skills location, for example:

```txt
<repo>/.agents/skills/tester/SKILL.md
<repo>/.agents/skills/commenter/SKILL.md
<repo>/.agents/skills/cleaner/SKILL.md
```

or:

```txt
~/.agents/skills/tester/SKILL.md
~/.agents/skills/commenter/SKILL.md
~/.agents/skills/cleaner/SKILL.md
```

Restart Codex if the skills do not appear immediately.
